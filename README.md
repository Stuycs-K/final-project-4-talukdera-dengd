# Project Description

This projectis a smaller version of the image steganography tool we covered in class: `stegsolve`, presenting the ability to view an image in various bit planes and color planes. 

Not only will users be able to look deeper into an image, they will also be able to draw on planes of their choosing, thereby altering the original image. 

Users will also be able to compress multiple images into one with our compression function.


# Features

* Show images in different planes
* Draw in bit planes to alter images
* Compress multiple images into one
